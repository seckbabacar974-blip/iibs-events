# IIBS Events

Site web des événements de l'Institut Informatique Business School de Dakar.

## C'est quoi ce projet ?

C'est mon projet de fin de semestre en L1 Génie Logiciel à l'IIBS. J'ai conçu et intégré un site web qui centralise tous les événements de l'école : hackathons, conférences, ateliers, journées portes ouvertes et plus encore.

## Fait par

Babacar Seck — L1 Génie Logiciel — IIBS 2025/2026

## Technologies utilisées

- HTML5
- Tailwind CSS
- JavaScript
- Font Awesome (icônes)
- Google Fonts (police Inter)

## Les pages du site

- Accueil
- Agenda des événements
- Actualités et Galerie
- Filières
- Contact
- Inscription
- Pages détail pour chaque événement
- Pages détail pour chaque filière
