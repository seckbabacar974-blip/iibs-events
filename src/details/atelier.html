description

Initiation aux failles web les plus courantes : injection SQL, XSS et CSRF. Les étudiants en Cyber Sécurité apprendront les bonnes pratiques pour sécuriser leurs applications dès la phase de développement. Niveau débutant, matériel fourni, 20 places disponibles.

Planning

09:00 Accueil & installation Mise en place des environnements de travail et présentation de l'animateur.
10:00 Théorie des failles web Présentation des vulnérabilités les plus courantes : SQL injection, XSS, CSRF.
11:30 Pratique guidée Exploitation de failles sur un environnement de test sécurisé.
13:00 Pause déjeuner Déjeuner et échanges entre participants.
14:30 Bonnes pratiques & correction Comment sécuriser son code dès le développement. Bilan et questions.
